 ### Endpoints

#### `GET /api/v1/hotels`
Retrieves all hotels.

#### `GET /api/v1/hotels/{id}`
Retrieves a specific hotel based on its ID.

#### `GET /api/v1/rooms`
Retrieves all rooms.

#### `GET /api/v1/rooms/{id}`
Retrieves a specific room based on its ID.

#### `GET /api/v1/users`
Retrieves all users.

#### `GET /api/v1/users/{id}`
Retrieves a specific user based on its ID.
